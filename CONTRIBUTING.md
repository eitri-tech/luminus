# Contributing to Luminus Daisy

First off, thank you for considering contributing to Luminus Daisy! It's people like you that make open source projects successful.

## How Can I Contribute?

### Reporting Bugs

If you encounter any bugs or issues while using Luminus Daisy, please report them by opening an issue in the [GitHub Issues](https://github.com/eitri-tech/luminus-daisy/issues) section. Before reporting a bug, please search existing issues to see if it's already been reported.

### Suggesting Enhancements

If you have an idea for an enhancement or new feature, feel free to open an issue in the [GitHub Issues](https://github.com/eitri-tech/luminus-daisy/issues) section. Provide as much detail as possible about the enhancement and the benefits it would bring.

### Submitting Pull Requests

1. **Fork the Repository**: Click the "Fork" button at the top right of the repository page and clone your fork locally.
   
2. **Create a Branch**: Create a new branch for your changes.
   ```sh
   git checkout -b my-feature-branch
   ```
   
3. **Make Changes**: Make your changes in the new branch. Ensure that your code follows the project's coding standards and includes relevant tests.

4. **Commit Changes**: Commit your changes with a clear and concise commit message.
   ```sh
   git commit -m "Add feature XYZ"
   ```

5. **Push to Fork**: Push your changes to your forked repository.
   ```sh
   git push origin my-feature-branch
   ```

6. **Open a Pull Request**: Go to the original repository and open a pull request with a detailed description of your changes.

### Code Style

- Ensure your code follows the project's coding standards.
- Include relevant tests for your changes.
- Run tests to ensure everything is working as expected.

### Documentation

If you are adding new features or changing functionality, please update the documentation accordingly. This includes updating any relevant markdown files or inline documentation.

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project, you agree to abide by its terms.

## Thank You

Thank you for your interest in contributing to Luminus Daisy. Your contributions are greatly appreciated!
